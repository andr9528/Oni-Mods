[h1]Critter Traits Reborn[/h1]

This mod adds traits to critters.
original mod made by Heinermann

[h1]Current latest Version: 1.0.0[/h1]

[hr][/hr]
Compatible with all modded critters.
Retroactively adds traits to existing critters

If you want to have a specific translation added, you can create a translation file with that template and send it to me via GitHub.
