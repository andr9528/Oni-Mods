[h1]Warning Notification Manager[/h1]

This mod allows to individually configure the warning notifications

[h1]Current latest Version: 1.0.0[/h1]

[hr][/hr]

For each of the following Notifications there are 3 settings:
[list][*] Mute Sound -> Removes the sound played when the notification triggers
[*]Pause Game -> will automatically pause the game when the notification triggers
[*]Pan Camera -> will move the camera to the trigger origin of the notification
[/list]

[h1]Configurable Notifications[/h1]
[list][*]Attack
[*]Starvation
[*]Made a mess
[*]Fleeing
[*]Suffocation
[*]High Stress
[*]Radiation Vomiting (DLC only)
[*]Scalding
[*]Entombed
[*]Incapacitated
[*]Plant has died
[/list]

[h2]Additional Settings[/h2]
[list][*]Suffocation Time Threshold: Define, how many seconds before death by suffocation the notification should trigger. Base value is 50s of breath remaining,
[*]Starvation Calorie Threshold: Amount of Calories at which the starvation notification triggers. Default is 1000kCal.
[/list]
